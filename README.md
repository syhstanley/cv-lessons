# 🖼️ Traditional Image Processing — Realtek 備戰課程

針對 Realtek 系統設計（Monitor/TV/Camera）面試與實習設計的傳統 IP 課程。

## 課程架構

| # | 主題 | 難度 |
|---|------|------|
| 01 | 色彩空間轉換（RGB / YUV / HSV） | ⭐ |
| 02 | 空間域濾波器（Gaussian / Median / Bilateral） | ⭐⭐ |
| 03 | 邊緣偵測（Sobel / Laplacian / Canny） | ⭐⭐ |
| 04 | ISP Pipeline（Demosaicing / White Balance / Gamma） | ⭐⭐⭐ |
| 05 | 頻域濾波（FFT / Low-pass / High-pass） | ⭐⭐⭐ |
| 06 | Noise Reduction（Spatial NR / Temporal NR） | ⭐⭐⭐ |
| 07 | 影像銳化與畫質增強（Unsharp Masking / Histogram EQ） | ⭐⭐ |
| 08 | Video 特有處理（Deinterlacing / Interpolation） | ⭐⭐⭐ |

## 環境需求

```bash
pip install opencv-python numpy matplotlib scipy
```
